# Veri_Yapilari-Syslog_Isleme_ve_Cift_Yonlu_Bagli_Liste_Uygulamasi
Bu proje, Linux işletim sistemindeki sistem günlüklerinin (syslog) okunarak, çift yönlü bağlı liste (doubly linked list) veri yapısı üzerinden bellekte dinamik olarak tutulmasını ve kronolojik/ters kronolojik olarak işlenmesini sağlayan bir C uygulamasıdır.
